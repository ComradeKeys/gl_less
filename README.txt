gl_less Version 1.0

GENERAL USAGE NOTES
--------------------------------------------------------------------------------
gl_less is a series of presentations created with LaTeX which implies a
basic knowledge of the C programming language. The aim of the lessons is to
take a person with no previous experience with graphics at all and evelate
them to an intermediate level of OpenGL. The course is full of demos for the
user to discover and apply to their toolbox of skills.
--------------------------------------------------------------------------------

gl_less table of contents
--------------------------------------------------------------------------------
1. Hello Triangle
2. Translations in OpenGL
3. Colors and Textures
4. Intro to 3D, Lighting and Normalized Vectors
5. Intro to GLSL
6. GLSL Part two.
7. Texturing a sphere
--------------------------------------------------------------------------------

INSTALLATION
--------------------------------------------------------------------------------
Normally the .pdf files are simply downloaded from the repository. However if
you would like to add your own modifications to my work feel free to do so that
adheres to the terms of the license. The .tex files can simply be compiled with
pdflatex with the following syntax:
$ pdflatex file.tex
